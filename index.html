Математика
Квадратні рівняння
Загальний вигляд: ax² + bx + c = 0, a ≠ 0

Дискримінант: D = b² − 4ac
Корені:
x₁ = (−b + √D) / 2a
x₂ = (−b − √D) / 2a

Якщо D > 0 — 2 корені,
D = 0 — 1 корінь,
D < 0 — дійсних коренів немає.
Математика
Формули Вієта
Для рівняння ax² + bx + c = 0 (a ≠ 0), корені x₁, x₂:

Сума коренів: x₁ + x₂ = −b / a
Добуток коренів: x₁ · x₂ = c / a

Якщо a = 1 (x² + px + q = 0):
x₁ + x₂ = −p,
x₁ · x₂ = q.
Математика
Тригонометрія
Основні означення:
sin α = протилежний катет / гіпотенуза
cos α = прилеглий катет / гіпотенуза
tg α = протилежний / прилеглий
ctg α = прилеглий / протилежний

Тотожності:
sin²α + cos²α = 1
1 + tg²α = 1 / cos²α
1 + ctg²α = 1 / sin²α
Математика
Геометрія
Піфагор: a² + b² = c²

Площа:
прямокутника S = a·b
трикутника S = (a·h)/2
круга S = πr²
кола L = 2πr

t — час (с)

Математика
Лінійні рівняння та нерівності
Рівняння: ax + b = 0 (a ≠ 0) ⇒ x = −b/a.

Нерівності:
якщо множиш / ділиш на від’ємне число — знак міняється.
(−)·( > ) ⇒ ( < )
Математика
Система 2 лінійних (Крамер)
a₁x + b₁y = c₁
a₂x + b₂y = c₂

Δ = a₁b₂ − a₂b₁
x = (c₁b₂ − c₂b₁) / Δ
y = (a₁c₂ − a₂c₁) / Δ
Математика
Степені, корені, модуль
aᵐ·aⁿ = aᵐ⁺ⁿ
aᵐ / aⁿ = aᵐ⁻ⁿ
(aᵐ)ⁿ = aᵐⁿ
√(a²) = |a|

|a| = a, якщо a ≥ 0; |a| = −a, якщо a < 0
Математика
Прогресії

aₙ = a₁ + (n−1)d
Sₙ = (a₁ + aₙ)·n / 2

Геометрична:
bₙ = b₁·qⁿ⁻¹
Sₙ = b₁( qⁿ − 1 ) / (q − 1), q ≠ 1

Логарифми

Тригонометрія
Табличні значення
Кути: 0° | 30° | 45° | 60° | 90°
sin: 0 | 1/2 | √2/2 | √3/2 | 1
cos: 1 | √3/2 | √2/2 | 1/2 | 0
tg: 0 | 1/√3 | 1 | √3 | —
ctg: — | √3 | 1 | 1/√3 | 0
Тригонометрія
Додавання / віднімання
sin(α ± β) = sinα·cosβ ± cosα·sinβ
cos(α ± β) = cosα·cosβ ∓ sinα·sinβ
tg(α ± β) = (tgα ± tgβ) / (1 ∓ tgα·tgβ)
ctg(α ± β) = (ctgα·ctgβ ∓ 1) / (ctgβ ± ctgα)
Тригонометрія
Подвійний кут
sin2α = 2sinα·cosα
cos2α = cos²α − sin²α = 1 − 2sin²α = 2cos²α − 1
tg2α = 2tgα / (1 − tg²α)
ctg2α = (ctg²α − 1) / (2ctgα)
Тригонометрія
Зведення + парність
sin(−α) = −sinα (непарна)
cos(−α) = cosα (парна)
tg(−α) = −tgα, ctg(−α) = −ctgα

sin(90°−α)=cosα, cos(90°−α)=sinα
tg(90°−α)=ctgα, ctg(90°−α)=tgα
