## 📰 Python редактор новостей

### Где найти:

**Основной файл:** `news_editor.py`
**Запуск:** `run_news_editor.bat` (на Windows) или `python news_editor.py`
**Инструкция:** `NEWS_EDITOR_README.md`

### Возможности:

✅ **Полное управление новостями:**
- ✨ Добавлять новые новости
- ✏️ Редактировать существующие
- 🗑️ Удалять ненужные
- 💾 Сохранять в JSON

✅ **Удобный интерфейс:**
- Двухпанельный редактор (список слева, редактор справа)
- Поиск по названию и дате
- Статус-бар с сообщениями
- Темная тема для удобства
- Подтверждение при удалении

✅ **Все поля новости:**
- Заголовок
- Дата публикации
- Основной текст
- Информация об авторе
- Флаги: "Важная", "Закрепить"
- Статус: "Опубликована" или "Черновик"
- Автоматический ID

### Как запустить:

1. **Windows - самый просто:**
   - Двойной клик на `run_news_editor.bat`

2. **Терминал:**
   ```
   python news_editor.py
   ```

3. **PowerShell:**
   ```
   python .\news_editor.py
   ```

## 🧪 Проверка мобильной адаптации

### В браузере:
1. Откройте сайт
2. Нажмите F12 (DevTools)
3. Нажмите Ctrl+Shift+M (мобильный вид)
4. Попробуйте разные размеры экранов
5. Проверьте что:
   - Все текст читается
   - Кнопки кликаются пальцем
   - Нет горизонтальной прокрутки
   - Изображения видны полностью

### Рекомендуемые размеры для тестирования:
- iPhone SE: 375x667
- iPhone 14: 390x844
- Samsung Galaxy A50: 540x720
- iPad Mini: 768x1024
- iPad Pro: 1024x1366
