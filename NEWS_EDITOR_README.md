# 📰 Редактор новостей KC-251

Приложение для управления новостями на сайте KC-251 с удобным графическим интерфейсом.

## 🚀 Запуск

### Вариант 1: Простой запуск (Windows)
```bash
python news_editor.py
```

или двойной клик на файл `news_editor.py`

### Вариант 2: Через PowerShell
```powershell
python .\news_editor.py
```

### Вариант 3: Создать ярлык (Windows)
1. Создайте файл `run_news_editor.bat`:
```batch
@echo off
cd /d %~dp0
python news_editor.py
pause
```
2. Сохраните его в папку проекта
3. Двойной клик для запуска

## 📋 Как использовать

### Интерфейс программы

**Левая часть** — Список новостей:
- 🔍 **Поиск** — найти новость по названию или дате
- **Список** — видны все новости (новые вверху)
- **➕ Новая** — создать новую новость
- **🗑️ Удалить** — удалить выбранную новость
- **💾 Сохранить** — сохранить все изменения в файл

**Правая часть** — Редактор новости:
- **Заголовок** — название новости
- **Дата** — дата публикации (YYYY-MM-DD)
- **Текст новости** — основной контент
- **Автор** — кто опубликовал
- **⭐ Важная новость** — отметить как важную
- **📌 Закрепить** — закрепить в верхней части списка
- **Статус** — "published" (опубликована) или "draft" (черновик)
- **✅ Сохранить новость** — применить изменения

## 🔄 Рабочий процесс

### Добавить новую новость:
1. Нажмите кнопку **➕ Новая**
2. Заполните поля справа
3. Нажмите **✅ Сохранить новость**
4. Нажмите **💾 Сохранить** внизу слева для записи в файл

### Отредактировать существующую:
1. Кликните на новость в списке
2. Измените нужные поля
3. Нажмите **✅ Сохранить новость**
4. Нажмите **💾 Сохранить** для записи в файл

### Удалить новость:
1. Кликните на новость в списке
2. Нажмите **🗑️ Удалить**
3. Подтвердите удаление
4. Нажмите **💾 Сохранить** для записи в файл

## 📁 Файлы

- **news.json** — основной файл с опубликованными новостями (автоматически обновляется)
- **news_history.json** — архив новостей (не используется редактором, но может быть полезен)

## ⚙️ Требования

- Python 3.6+
- tkinter (обычно идет с Python)

На Windows `tkinter` часто установлен по умолчанию.

Если возникает ошибка `ModuleNotFoundError: No module named 'tkinter'`:
- Windows: переинсталлируйте Python с опцией "tcl/tk and IDLE"
- Linux: `sudo apt-get install python3-tk`
- macOS: должно быть в стандартной установке

## 💡 Советы

- Всегда нажимайте **💾 Сохранить** перед закрытием программы
- Дата автоматически заполняется текущей датой при создании новости
- ID новости генерируется автоматически и не может быть изменен
- Поиск работает по заголовку и дате
- Новости в файле автоматически сортируются по дате (новые в начале)

## 🐛 Исправление проблем

**Ошибка "Файл news.json не найден":**
- Убедитесь, что программа запущена из той же папки, где находится news.json

**Новости не сохраняются:**
- Проверьте, что файл news.json не открыт в другом приложении
- Проверьте права доступа на запись в папку

**Ошибка при запуске:**
- Убедитесь, что установлен Python 3.6+
- Проверьте установку tkinter (см. выше)
